Warehouse Inventory Management System


* This is a Java-based system to manage warehouse items using ArrayList.

Features We can add
- Add Item
- Remove Item
- Update Quantity
- Search Item
- Display Items
- Exit

 How to Run :

1. Compile:
   javac Warehouse.java

2. Run:
   java Warehouse
